How To Create A Website Using GitHub

Step 1.

On the Github Website in the top right corner click on the + icon and select New Repository

Step 2.

In the create a Repository menu name your repository to your liking and select the "Initialize this repository with a README" check box and create your Repository

Step 3.

on the newly created repository page select the upload files option to the right of the green box

Step 4.

Select all the source files you wish to upload and simply drag and drop into the "Drag Files Here" box

Step 5.

Once files are placed and uploaded select the "Commit Changes" option

Step 6.

Select the Settings Tab

Step 7.

Scroll to the bottom of the page and select the dropdown menu on "None" and select the "Master Branch" option

Step 8.

Wait for a highlighted Green box to appear over your published site url and click the link

Step 9.

Your Site is now published congratulations
